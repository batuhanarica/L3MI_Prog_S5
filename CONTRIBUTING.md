# Contributing to L3MI Programming Repository

## Adding Google Colab Notebooks from Your Phone

Yes! You can add Google Colab notebooks from your phone. Here are several methods:

### Method 1: Using Google Colab Mobile App

1. Open [Google Colab](https://colab.research.google.com) in your mobile browser
2. Create or open your notebook
3. Click on the menu (three dots) → **File** → **Save a copy in GitHub**
4. Select this repository: `batuhanarica/L3MI_Prog_S5`
5. Choose the branch (or create a new one)
6. Specify the file path: `notebooks/your_notebook_name.ipynb`
7. Add a commit message
8. Click **OK** to commit

### Method 2: Using GitHub Mobile App

1. Open your notebook in Google Colab
2. Download the `.ipynb` file to your phone
3. Open GitHub mobile app
4. Navigate to this repository
5. Go to the `notebooks/` folder
6. Tap the **+** button → **Upload files**
7. Select your downloaded `.ipynb` file
8. Add a commit message and commit

### Method 3: Using GitHub Web Interface on Mobile

1. Download your `.ipynb` notebook from Colab to your phone
2. Open [GitHub.com](https://github.com/batuhanarica/L3MI_Prog_S5) in your mobile browser
3. Navigate to the `notebooks/` folder
4. Tap **Add file** → **Upload files**
5. Select your `.ipynb` file
6. Commit the changes

## File Organization

- Place all Colab notebooks in the `notebooks/` directory
- Use descriptive names for your notebooks, e.g., `exercise_3_2.ipynb`
- Include comments and markdown cells to explain your work

## Guidelines

- Keep notebooks well-documented with markdown cells
- Include clear titles and descriptions
- Test your code before committing
- Use meaningful commit messages

## Questions?

If you have any questions about contributing, feel free to open an issue!
