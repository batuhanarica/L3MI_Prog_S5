# L3MI Programming - Semester 5

This repository contains programming exercises and projects for L3 Mathematics and Computer Science, Semester 5.

## Contents

- **C Programming**: SAT solver implementations and programming exercises
  - `sat_q1.c` - SAT solver question 1
  - `3.2.c` - Exercise 3.2

- **Colab Notebooks**: Yes, you can add Google Colab notebooks to this repository! 
  - Store notebooks in the `notebooks/` directory
  - Colab notebooks can be added from any device, including mobile phones

## Adding Colab Notebooks

You can add Google Colab notebooks from your phone or any device. See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.
